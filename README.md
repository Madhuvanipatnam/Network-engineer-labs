# 📡 DHCP Configuration Lab (Packet Tracer)

## 📌 Overview
The project demonstrates how DHCP (Dynamic Host Configuration Protocol) works using three different network topologies in Cisco Packet Tracer.

---

## 🧪 Topologies Covered

### 🔹 Topology 1: DHCP Server (Same Network)
- Dedicated server assigns IP addresses
- All devices are in the same subnet
- No router required

---

### 🔹 Topology 2: Router as DHCP Server
- Router acts as both:
  - Default Gateway
  - DHCP Server
- Simplified and commonly used in real networks

---

### 🔹 Topology 3: DHCP Relay (Different Networks)
- DHCP server is in a different subnet
- Uses `ip helper-address`
- Demonstrates real-world enterprise setup

---

## 🛠️ Tools Used
- Cisco Packet Tracer
